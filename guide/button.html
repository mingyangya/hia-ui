HiaButton 按钮

常用的操作按钮。

基础用法

基础的按钮用法。

使用 type 、 plain 、 round 和 circle 属性来定义 Button 的样式。

<template>
  <div class="source-code-body">
    <HiaButtonGroup>
      <HiaButton>默认按钮</HiaButton>
      <HiaButton type="primary">主要按钮</HiaButton>
      <HiaButton type="success">成功按钮</HiaButton>
      <HiaButton type="info">信息按钮</HiaButton>
      <HiaButton type="warning">警告按钮</HiaButton>
      <HiaButton type="danger">危险按钮</HiaButton>
    </HiaButtonGroup>

    <HiaButtonGroup>
      <HiaButton plain>朴素按钮</HiaButton>
      <HiaButton type="primary" plain>主要按钮</HiaButton>
      <HiaButton type="success" plain>成功按钮</HiaButton>
      <HiaButton type="info" plain>信息按钮</HiaButton>
      <HiaButton type="warning" plain>警告按钮</HiaButton>
      <HiaButton type="danger" plain>危险按钮</HiaButton>
    </HiaButtonGroup>

    <HiaButtonGroup>
      <HiaButton round>圆角按钮</HiaButton>
      <HiaButton type="primary" round>主要按钮</HiaButton>
      <HiaButton type="success" round>成功按钮</HiaButton>
      <HiaButton type="info" round>信息按钮</HiaButton>
      <HiaButton type="warning" round>警告按钮</HiaButton>
      <HiaButton type="danger" round>危险按钮</HiaButton>
    </HiaButtonGroup>

    <HiaButtonGroup>
      <HiaButton icon="el-icon-search" circle></HiaButton>
      <HiaButton type="primary" icon="el-icon-edit" circle></HiaButton>
      <HiaButton type="success" icon="el-icon-check" circle></HiaButton>
      <HiaButton type="info" icon="el-icon-message" circle></HiaButton>
      <HiaButton type="warning" icon="el-icon-star-off" circle></HiaButton>
      <HiaButton type="danger" icon="el-icon-delete" circle></HiaButton>
    </HiaButtonGroup>
  </div>
</template>

Attribute

属性 类型 默认值 描述
type string 'default' 按钮类型
size string - 按钮大小
icon string '' 图标
nativeType string 'button' 原生按钮类型
loading bool - 是否加载中
disabled bool - 是否禁用
plain bool - 是否朴素按钮
autofocus bool - 是否自动聚焦
round bool - 是否圆角按钮
circle bool - 是否圆形按钮
